Vamos começar? Antes de mais nada, um overview do que iremos tratar no Workshop de hoje. Vamos em todo o workshop utilizar uma única aplicação de "Nome a definir ainda" e que será completamente feita com Next.Js.

Se você não consegue esperar, já pode ir olhando no nosso Figma para ver a aplicação que será construída. O link é esse.

Os principais tópicos que iremos ver são:
- **App Router**: Entender a nova estrutura de pastas e como funcionam as rotas estáticas e dinâmicas.
- **Layouts**: Como criar layouts reutilizáveis e aninhados para diferentes seções do site.
- **Server e Client Components**: Diferenças, quando usar cada um e como eles interagem.
- **API Routes**: Criar endpoints de API diretamente no Next.js.
- **Data Fetching**: Técnicas para buscar dados no servidor e no cliente.
- **Deploy**: Publicar a aplicação em plataformas como Vercel ou Netlify.
