---
title: Pré-requisitos
description: Pré-requisitos para participar do workshop de Next.js para devs Node.js.
---
Para aproveitar ao máximo o workshop de **Next.js** versãos 15+, é importante que você tenha alguns conhecimentos prévios e ferramentas instaladas. Aqui estão os pré-requisitos recomendados:
- **Conhecimentos básicos de JavaScript e Node.js**: Familiaridade com conceitos como módulos, promessas e async/await.
- **Noções de React**: Entender componentes, props, estado e hooks básicos.
- **Ferramentas instaladas**:
  - Node.js (versão LTS recomendada, como 22.x)
  - Docker e Docker Compose (opcional, mas recomendado para ambientes isolados)
  - Um gerenciador de pacotes (npm ou pnpm)
  - Editor de código (recomendamos VSCode ou Vim)
- **Conhecimentos básicos de terminal/linha de comando**: Saber navegar entre pastas, criar arquivos e executar comandos simples.
- **Noções de HTTP e APIs REST**: Entender métodos HTTP comuns (GET, POST, etc.) e conceitos básicos de APIs.
- **Conta em plataformas de deploy**: Ter uma conta no Vercel ou Netlify para publicar a aplicação ao final do workshop.
Com esses pré-requisitos atendidos, você estará pronto para acompanhar o conteúdo e participar ativamente do workshop!
