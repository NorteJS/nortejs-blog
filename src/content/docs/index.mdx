## Como usar este material

Este repo nasce para apoiar a comunidade **NorteJS** com um roteiro enxuto de palestra + workshop. Cada seção é pensada para:

- introduzir o conceito de forma direta;
- mostrar um exemplo real em TypeScript;
- propor pequenas interações para usar em sala ou em live coding.
